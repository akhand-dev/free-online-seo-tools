Markdown Cheat Sheet — Complete Syntax Guide

Published on May 18, 2026 • 7 min read
[Ad Space]

Markdown is the go-to formatting language for README files, documentation, blogs, forums, and note-taking apps. This cheat sheet covers every syntax element you'll ever need.

Headings

# Heading 1
## Heading 2
### Heading 3
#### Heading 4

Text Formatting

**Bold text**
*Italic text*
~~Strikethrough~~
**_Bold and italic_**
`Inline code`

Links & Images

[Link Text](https://example.com)
![Alt Text](image-url.jpg)
[Link with title](https://example.com "Title")

Lists

Unordered List

- Item 1
- Item 2
  - Nested item
  - Another nested item
- Item 3

Ordered List

1. First item
2. Second item
3. Third item
[Ad Space]

Blockquotes

> This is a blockquote.
> It can span multiple lines.
>> Nested blockquote

Code Blocks

```javascript
function hello() {
    console.log("Hello, World!");
}
```

Tables

| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Cell 1   | Cell 2   | Cell 3   |
| Cell 4   | Cell 5   | Cell 6   |

Horizontal Rule

---

Task Lists

- [x] Completed task
- [ ] Incomplete task
- [ ] Another task
